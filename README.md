# My Dotfiles
- WE: BSPWM & i3
- Status Bar: Polybar
- Launcher: Rofi
- Editor: Neovim(LunarVim) & VSCodium

## Dependecies
- feh
- scrot
- xidlehook
- i3lock-color
- FiraCode Nerd Font

## Scripts
- ./power = Power menu
- ./scrot & ./scrot5 = Screenshotter
- ./web-search.sh = Web search
- ./lock.sh = i3lock config
- ./idle = idle manager
- ./launchp & ./launchb = polybar launchers
- ./i3lock = i3lock-color
- ./i3hint = display i3 config
- ./check.sh = toggle launcher terminal app
